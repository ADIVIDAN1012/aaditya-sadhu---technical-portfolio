# Aaditya Sadhu | Technical & Systems Portfolio

A formal synthesis of 6th-semester Information Technology research and development, showcasing complex systems, intelligence frameworks, and mathematical computational tools.

## Projects Covered
- **Beacon**: Custom Human-Centric Programming Language (C/Python Compiler).
- **Combatlas**: AI-Driven Defense Encyclopedia (Llama 3 / Groq).
- **Matoolox**: Mathematical Toolbox & Academic Submission Pipeline.

## Technical Stack
- **Frontend**: React 18, Vite, TypeScript, Framer Motion, Lucide React.
- **Backend/Systems**: C, Python, Llama-3 (Groq Integration).

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Configuration**:
   Set your `VITE_GROQ_API_KEY` in `.env.local`.
3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
